# AquaConnect - Gestion des Demandes d'Aide Technique

AquaConnect est une application de gestion complète pour les demandes d'aide technique dans le domaine de l'assainissement. Cette application permet de gérer les clients, les demandes, les devis, les articles et les ordres de travaux.

## Fonctionnalités principales

- **Gestion des clients** : Création, modification et consultation des fiches clients
- **Gestion des demandes** : Suivi des demandes d'aide technique de l'émission à la clôture
- **Gestion des devis** : Création de devis détaillés avec calculs automatiques
- **Gestion des articles** : Catalogue complet des articles avec familles, unités et spécifications
- **Gestion des ordres de travaux** : Suivi des interventions sur le terrain
- **Tableau de bord** : Vue d'ensemble des statistiques et indicateurs clés
- **Gestion des utilisateurs** : Administration des comptes et des rôles
- **Paramètres** : Configuration de l'application et des préférences

## Technologies utilisées

- **Frontend** : React.js avec Tailwind CSS
- **Backend** : Node.js avec Express
- **Base de données** : SQL Server
- **Authentification** : JWT (JSON Web Tokens)
- **Styling** : OBAT-style design avec thème sombre/clair

## Installation

1. Cloner le repository
2. Installer les dépendances frontend et backend
3. Configurer la base de données SQL Server
4. Exécuter les scripts d'initialisation
5. Démarrer les serveurs frontend et backend
